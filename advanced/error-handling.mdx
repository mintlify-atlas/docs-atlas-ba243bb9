## Overview

Lua uses a exception-based error handling system implemented with long jumps (or C++ exceptions when compiled as C++). Errors can be caught and handled at different levels using protected calls.

## Error Status Codes

Lua defines several error status codes in `lua.h:48`:

```c
#define LUA_OK        0  /* No errors */
#define LUA_YIELD     1  /* Thread yielded */
#define LUA_ERRRUN    2  /* Runtime error */
#define LUA_ERRSYNTAX 3  /* Syntax error during precompilation */
#define LUA_ERRMEM    4  /* Memory allocation error */
#define LUA_ERRERR    5  /* Error while running message handler */
```

## Exception Handling Mechanism

Lua uses platform-specific exception mechanisms defined in `ldo.c:74`:

```c
#if defined(__cplusplus) && !defined(LUA_USE_LONGJMP)
  /* C++ exceptions */
  #define LUAI_THROW(L,c)    throw(c)
#elif defined(LUA_USE_POSIX)
  /* POSIX _longjmp/_setjmp (more efficient) */
  #define LUAI_THROW(L,c)    _longjmp((c)->b, 1)
  #define LUAI_TRY(L,c,f,ud) if (_setjmp((c)->b) == 0) ((f)(L, ud))
#else
  /* ISO C longjmp/setjmp */
  #define LUAI_THROW(L,c)    longjmp((c)->b, 1)
  #define LUAI_TRY(L,c,f,ud) if (setjmp((c)->b) == 0) ((f)(L, ud))
#endif
```

<Note>
When compiled as C++, Lua uses C++ exceptions for better integration with C++ code. Otherwise, it uses `setjmp`/`longjmp` for exception handling.
</Note>

## Long Jump Buffers

Lua maintains a chain of error handlers (`ldo.c:61`):

```c
typedef struct lua_longjmp {
  struct lua_longjmp *previous;
  jmp_buf b;
  volatile TStatus status;  /* error code */
} lua_longjmp;
```

## Throwing Errors

### From C API

```c
/* Raise an error with a string message */
luaL_error(L, "error message: %s", details);

/* Raise a memory error */
luaM_error(L);

/* Generic error throw */
luaD_throw(L, LUA_ERRRUN);
```

### Error Throw Implementation

From `ldo.c:125`:

```c
l_noret luaD_throw (lua_State *L, TStatus errcode) {
  if (L->errorJmp) {  /* thread has an error handler? */
    L->errorJmp->status = errcode;  /* set status */
    LUAI_THROW(L, L->errorJmp);  /* jump to it */
  }
  else {  /* thread has no error handler */
    global_State *g = G(L);
    lua_State *mainth = mainthread(g);
    errcode = luaE_resetthread(L, errcode);  /* close all upvalues */
    L->status = errcode;
    if (mainth->errorJmp) {  /* main thread has a handler? */
      setobjs2s(L, mainth->top.p++, L->top.p - 1);  /* copy error obj. */
      luaD_throw(mainth, errcode);  /* re-throw in main thread */
    }
    else {  /* no handler at all; abort */
      if (g->panic) {  /* panic function? */
        lua_unlock(L);
        g->panic(L);  /* call panic function (last chance to jump out) */
      }
      abort();
    }
  }
}
```

<Tip>
If there's no error handler and no panic function, Lua calls `abort()` to terminate the program. Always set a panic function for embedded applications.
</Tip>

## Protected Calls

### lua_pcall

Executes a function in protected mode:

```c
int lua_pcall(lua_State *L, int nargs, int nresults, int msgh);
```

**Parameters:**
- `nargs`: Number of arguments on the stack
- `nresults`: Number of expected results (or `LUA_MULTRET`)
- `msgh`: Stack index of message handler function (or 0)

**Returns:** Status code (`LUA_OK`, `LUA_ERRRUN`, `LUA_ERRMEM`, `LUA_ERRERR`)

```c
lua_getglobal(L, "riskyfunction");
lua_pushstring(L, "arg");
if (lua_pcall(L, 1, 1, 0) != LUA_OK) {
  /* Error occurred - error message is on stack */
  const char *error = lua_tostring(L, -1);
  fprintf(stderr, "Error: %s\n", error);
  lua_pop(L, 1);
  return -1;
}
/* Result is on stack */
```

### Protected Call Implementation

From `ldo.c:1089`:

```c
TStatus luaD_pcall (lua_State *L, Pfunc func, void *u, ptrdiff_t old_top,
                                  ptrdiff_t ef) {
  TStatus status;
  CallInfo *old_ci = L->ci;
  lu_byte old_allowhooks = L->allowhook;
  ptrdiff_t old_errfunc = L->errfunc;
  L->errfunc = ef;
  status = luaD_rawrunprotected(L, func, u);
  if (l_unlikely(status != LUA_OK)) {  /* an error occurred? */
    L->ci = old_ci;
    L->allowhook = old_allowhooks;
    status = luaD_closeprotected(L, old_top, status);
    luaD_seterrorobj(L, status, restorestack(L, old_top));
    luaD_shrinkstack(L);   /* restore stack size in case of overflow */
  }
  L->errfunc = old_errfunc;
  return status;
}
```

## Message Handler

A message handler function can transform error messages:

```c
/* Push message handler */
lua_pushcfunction(L, traceback);
int msgh = lua_gettop(L);

/* Push function and arguments */
lua_getglobal(L, "func");
lua_pushstring(L, "arg");

/* Call with message handler */
if (lua_pcall(L, 1, 1, msgh) != LUA_OK) {
  /* Error message was processed by traceback() */
  fprintf(stderr, "%s\n", lua_tostring(L, -1));
  lua_pop(L, 1);
}
lua_remove(L, msgh);
```

Example message handler with stack trace:

```c
static int traceback(lua_State *L) {
  const char *msg = lua_tostring(L, 1);
  if (msg == NULL) {  /* is error object not a string? */
    if (luaL_callmeta(L, 1, "__tostring") &&  /* does it have a metamethod */
        lua_type(L, -1) == LUA_TSTRING)  /* that produces a string? */
      return 1;  /* that is the message */
    else
      msg = lua_pushfstring(L, "(error object is a %s value)",
                               luaL_typename(L, 1));
  }
  luaL_traceback(L, L, msg, 1);  /* append a standard traceback */
  return 1;  /* return the traceback */
}
```

## Stack Unwinding

When an error occurs, Lua unwinds the call stack, closing upvalues and calling `__close` metamethods.

From `ldo.c:874`:

```c
static void unroll (lua_State *L, void *ud) {
  CallInfo *ci;
  UNUSED(ud);
  while ((ci = L->ci) != &L->base_ci) {  /* something in the stack */
    if (!isLua(ci))  /* C function? */
      finishCcall(L, ci);  /* complete its execution */
    else {  /* Lua function */
      luaV_finishOp(L);  /* finish interrupted instruction */
      luaV_execute(L, ci);  /* execute down to higher C 'boundary' */
    }
  }
}
```

## Error Recovery

### Finding Protected Calls

From `ldo.c:892`:

```c
static CallInfo *findpcall (lua_State *L) {
  CallInfo *ci;
  for (ci = L->ci; ci != NULL; ci = ci->previous) {  /* search for a pcall */
    if (ci->callstatus & CIST_YPCALL)
      return ci;
  }
  return NULL;  /* no pending pcall */
}
```

### Precover Function

From `ldo.c:963`:

```c
static TStatus precover (lua_State *L, TStatus status) {
  CallInfo *ci;
  while (errorstatus(status) && (ci = findpcall(L)) != NULL) {
    L->ci = ci;  /* go down to recovery functions */
    setcistrecst(ci, status);  /* status to finish 'pcall' */
    status = luaD_rawrunprotected(L, unroll, NULL);
  }
  return status;
}
```

## Setting Error Objects

From `ldo.c:112`:

```c
void luaD_seterrorobj (lua_State *L, TStatus errcode, StkId oldtop) {
  if (errcode == LUA_ERRMEM) {  /* memory error? */
    setsvalue2s(L, oldtop, G(L)->memerrmsg); /* reuse preregistered msg. */
  }
  else {
    lua_assert(errorstatus(errcode));  /* must be a real error */
    lua_assert(!ttisnil(s2v(L->top.p - 1)));  /* with a non-nil object */
    setobjs2s(L, oldtop, L->top.p - 1);  /* move it to 'oldtop' */
  }
  L->top.p = oldtop + 1;  /* top goes back to old top plus error object */
}
```

<Note>
Memory errors use a pre-allocated error message to avoid allocation during error handling.
</Note>

## Panic Function

Set a panic function to handle errors outside protected contexts:

```c
static int panic_handler(lua_State *L) {
  const char *msg = lua_tostring(L, -1);
  fprintf(stderr, "PANIC: unprotected error in call to Lua API (%s)\n",
          msg ? msg : "error object is not a string");
  return 0;  /* return to Lua to abort */
}

lua_State *L = luaL_newstate();
lua_atpanic(L, panic_handler);
```

## Error Handling in Lua

### pcall

```lua
local status, result = pcall(function()
  -- Code that might error
  error("something went wrong")
end)

if not status then
  print("Error:", result)
else
  print("Success:", result)
end
```

### xpcall with Message Handler

```lua
local function error_handler(err)
  return debug.traceback("Error: " .. tostring(err), 2)
end

local status, result = xpcall(function()
  error("something went wrong")
end, error_handler)

if not status then
  print(result)  -- Includes stack trace
end
```

### assert

```lua
-- Raises an error if condition is false
local value = assert(io.open("file.txt"), "Failed to open file")
```

## Best Practices

1. **Always use pcall** for operations that might fail (file I/O, parsing, etc.)
2. **Provide context** in error messages with relevant details
3. **Use message handlers** to add stack traces for debugging
4. **Handle memory errors** specially - they can't allocate new strings
5. **Set a panic function** in embedded applications
6. **Clean up resources** before throwing errors

## Stack Overflow Protection

Lua protects against stack overflow (`ldo.c:361`):

```c
int luaD_growstack (lua_State *L, int n, int raiseerror) {
  int size = stacksize(L);
  if (l_unlikely(size > MAXSTACK)) {
    /* Stack is larger than maximum */
    lua_assert(stacksize(L) == ERRORSTACKSIZE);
    if (raiseerror)
      luaD_errerr(L);  /* stack error inside message handler */
    return 0;
  }
  // ...
  if (raiseerror)
    luaG_runerror(L, "stack overflow");
  return 0;
}
```

<Tip>
The maximum stack size is typically 1,000,000 slots (`LUAI_MAXSTACK` in `ldo.c:192`). Deep recursion will trigger a stack overflow error.
</Tip>
