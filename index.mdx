## Why Lua?

Lua is a powerful, efficient, lightweight, embeddable scripting language. It supports procedural programming, object-oriented programming, functional programming, data-driven programming, and data description.

<CardGroup cols={2}>
  <Card title="Lightweight" icon="feather">
    Small footprint - the entire Lua interpreter compiles to under 300KB
  </Card>
  <Card title="Fast" icon="bolt">
    One of the fastest scripting languages available, with optional JIT compilation
  </Card>
  <Card title="Embeddable" icon="puzzle-piece">
    Simple C API makes it easy to embed Lua in any application
  </Card>
  <Card title="Portable" icon="globe">
    Written in ANSI C, runs on all platforms with a C compiler
  </Card>
</CardGroup>

## Get Started

<Steps>
  <Step title="Install Lua">
    Download and install Lua on your system
    
    <CodeGroup>
    ```bash Ubuntu/Debian
    sudo apt-get install lua5.4
    ```
    
    ```bash macOS
    brew install lua
    ```
    
    ```bash From Source
    curl -R -O http://www.lua.org/ftp/lua-5.4.7.tar.gz
    tar zxf lua-5.4.7.tar.gz
    cd lua-5.4.7
    make all test
    ```
    </CodeGroup>
  </Step>
  
  <Step title="Write Your First Program">
    Create a simple Lua script
    
    ```lua hello.lua
    print("Hello, Lua!")
    
    -- Variables and expressions
    local name = "World"
    print("Hello, " .. name .. "!")
    
    -- Functions
    function greet(person)
      return "Hello, " .. person .. "!"
    end
    
    print(greet("Lua"))
    ```
  </Step>
  
  <Step title="Run Your Program">
    Execute your Lua script
    
    ```bash
    lua hello.lua
    ```
  </Step>
</Steps>

## Quick Example

Here's a taste of what Lua can do:

```lua example.lua lines
-- Factorial function
function factorial(n)
  if n == 0 then
    return 1
  else
    return n * factorial(n - 1)
  end
end

-- Tables as arrays
local numbers = {1, 2, 3, 4, 5}
for i, v in ipairs(numbers) do
  print(string.format("%d! = %d", v, factorial(v)))
end

-- Tables as dictionaries
local person = {
  name = "John",
  age = 30,
  greet = function(self)
    print("Hello, I'm " .. self.name)
  end
}

person:greet()
```
