## Library Design

Most Lua libraries provide their functions in two ways:

1. **As module functions** - Called with dot notation (e.g., `string.upper(s)`)
2. **As methods** - Called with colon notation on objects (e.g., `s:upper()`)

The string library, for example, sets its table as the metatable for all strings, allowing both calling styles.
