## What is Lua?

Lua is a powerful, efficient, lightweight, embeddable scripting language. It supports procedural programming, object-oriented programming, functional programming, data-driven programming, and data description.

<Note>
Lua is developed by a team at PUC-Rio, the Pontifical Catholic University of Rio de Janeiro in Brazil. The language was created in 1993 by Roberto Ierusalimschy, Luiz Henrique de Figueiredo, and Waldemar Celes.
</Note>

## Core Philosophy

Lua is designed with several key principles in mind:

- **Simplicity**: Simple, clean syntax that's easy to learn and read
- **Efficiency**: Fast execution and low memory footprint
- **Portability**: Runs on virtually any platform
- **Embeddability**: Easy to integrate into applications via C API
- **Power**: Provides powerful mechanisms like first-class functions and proper tail calls

## Who Uses Lua?

Lua is used in a wide variety of applications:

- **Game Development**: Popular in game engines and scripting
- **Embedded Systems**: Network equipment, industrial programs
- **Web Development**: Nginx with OpenResty, Apache with mod_lua
- **Application Scripting**: Adobe Lightroom, VLC media player, and many more
