## Overview

The Lua debug interface provides a comprehensive set of functions for introspecting and debugging Lua programs. It allows you to examine the call stack, access local variables and upvalues, retrieve function information, and implement profilers and debuggers.

## lua_Debug Structure

The `lua_Debug` structure holds debug information about a function or activation record.

```c
struct lua_Debug {
  int event;
  const char *name;           /* (n) */
  const char *namewhat;       /* (n) 'global', 'local', 'field', 'method' */
  const char *what;           /* (S) 'Lua', 'C', 'main', 'tail' */
  const char *source;         /* (S) */
  size_t srclen;              /* (S) */
  int currentline;            /* (l) */
  int linedefined;            /* (S) */
  int lastlinedefined;        /* (S) */
  unsigned char nups;         /* (u) number of upvalues */
  unsigned char nparams;      /* (u) number of parameters */
  char isvararg;              /* (u) */
  unsigned char extraargs;    /* (t) number of extra arguments */
  char istailcall;            /* (t) */
  int ftransfer;              /* (r) index of first value transferred */
  int ntransfer;              /* (r) number of transferred values */
  char short_src[LUA_IDSIZE]; /* (S) */
  /* private part */
  struct CallInfo *i_ci;      /* active function */
};
```

<ParamField path="event" type="int">
  Event code that triggered the hook (for hooks only)
</ParamField>

<ParamField path="name" type="const char*">
  A reasonable name for the function (requires 'n' option)
</ParamField>

<ParamField path="namewhat" type="const char*">
  Explains the name field: 'global', 'local', 'field', 'method', or empty string
</ParamField>

<ParamField path="what" type="const char*">
  'Lua' for Lua functions, 'C' for C functions, 'main' for the main chunk
</ParamField>

<ParamField path="source" type="const char*">
  Source of the chunk that created the function
</ParamField>

<ParamField path="currentline" type="int">
  Current line where the function is executing (requires 'l' option)
</ParamField>

<ParamField path="nups" type="unsigned char">
  Number of upvalues (requires 'u' option)
</ParamField>

<ParamField path="nparams" type="unsigned char">
  Number of parameters (requires 'u' option)
</ParamField>

<ParamField path="isvararg" type="char">
  1 if function is variadic, 0 otherwise (requires 'u' option)
</ParamField>

<ParamField path="istailcall" type="char">
  1 if this function invocation was called by a tail call (requires 't' option)
</ParamField>

## Core Functions

### lua_getstack

Retrieves information about the runtime stack.

```c
int lua_getstack(lua_State *L, int level, lua_Debug *ar);
```

<ParamField path="L" type="lua_State*">
  The Lua state
</ParamField>

<ParamField path="level" type="int">
  Stack level (0 is the current function, 1 is the function that called it, etc.)
</ParamField>

<ParamField path="ar" type="lua_Debug*">
  Pointer to a lua_Debug structure to be filled with stack information
</ParamField>

**Returns:** 1 if the level is valid, 0 otherwise.

**Example:**

```c
lua_Debug ar;
if (lua_getstack(L, 0, &ar)) {
    lua_getinfo(L, "nSl", &ar);
    printf("Function: %s\n", ar.name ? ar.name : "<unknown>");
    printf("Line: %d\n", ar.currentline);
}
```

### lua_getinfo

Returns information about a specific function or function invocation.

```c
int lua_getinfo(lua_State *L, const char *what, lua_Debug *ar);
```

<ParamField path="L" type="lua_State*">
  The Lua state
</ParamField>

<ParamField path="what" type="const char*">
  A string describing what information to retrieve. Can contain:
  - 'n': fills in `name` and `namewhat`
  - 'S': fills in `source`, `short_src`, `linedefined`, `lastlinedefined`, and `what`
  - 'l': fills in `currentline`
  - 'u': fills in `nups`, `nparams`, and `isvararg`
  - 't': fills in `istailcall` and `extraargs`
  - 'r': fills in `ftransfer` and `ntransfer`
  - 'L': pushes a table with valid lines for the function
  - 'f': pushes the function onto the stack
  - '>': function is on top of stack (not from a stack level)
</ParamField>

<ParamField path="ar" type="lua_Debug*">
  Pointer to lua_Debug structure (from lua_getstack or with function on stack)
</ParamField>

**Returns:** 1 on success, 0 on error.

**Example:**

```c
// Get info about function on stack
lua_Debug ar;
if (lua_getstack(L, 1, &ar)) {
    lua_getinfo(L, "nSlu", &ar);
    printf("Called from: %s\n", ar.short_src);
    printf("Function: %s (%s)\n", ar.name ? ar.name : "?", ar.what);
    printf("Line: %d\n", ar.currentline);
    printf("Parameters: %d\n", ar.nparams);
}

// Get info about function object
lua_pushcfunction(L, my_function);
lua_getinfo(L, ">Su", &ar);
printf("Upvalues: %d\n", ar.nups);
```

### lua_getlocal

Gets information about a local variable or parameter.

```c
const char *lua_getlocal(lua_State *L, const lua_Debug *ar, int n);
```

<ParamField path="L" type="lua_State*">
  The Lua state
</ParamField>

<ParamField path="ar" type="const lua_Debug*">
  Activation record from lua_getstack, or NULL to inspect function parameters
</ParamField>

<ParamField path="n" type="int">
  Local variable index (1 is the first local)
</ParamField>

**Returns:** Name of the local variable, or NULL if index is out of range. Pushes the variable's value onto the stack.

**Example:**

```c
lua_Debug ar;
if (lua_getstack(L, 0, &ar)) {
    int i = 1;
    const char *name;
    while ((name = lua_getlocal(L, &ar, i)) != NULL) {
        printf("Local %d: %s = ", i, name);
        // Value is now on top of stack
        if (lua_isstring(L, -1))
            printf("%s\n", lua_tostring(L, -1));
        else
            printf("<%s>\n", lua_typename(L, lua_type(L, -1)));
        lua_pop(L, 1);  // Remove value
        i++;
    }
}
```

### lua_setlocal

Sets the value of a local variable.

```c
const char *lua_setlocal(lua_State *L, const lua_Debug *ar, int n);
```

<ParamField path="L" type="lua_State*">
  The Lua state
</ParamField>

<ParamField path="ar" type="const lua_Debug*">
  Activation record from lua_getstack
</ParamField>

<ParamField path="n" type="int">
  Local variable index (1 is the first local)
</ParamField>

**Returns:** Name of the local variable, or NULL if index is out of range. Pops the new value from the stack.

**Example:**

```c
lua_Debug ar;
if (lua_getstack(L, 0, &ar)) {
    lua_pushstring(L, "new value");
    const char *name = lua_setlocal(L, &ar, 1);
    if (name)
        printf("Set local '%s'\n", name);
}
```

### lua_getupvalue

Gets information about a closure's upvalue.

```c
const char *lua_getupvalue(lua_State *L, int funcindex, int n);
```

<ParamField path="L" type="lua_State*">
  The Lua state
</ParamField>

<ParamField path="funcindex" type="int">
  Stack index of the function
</ParamField>

<ParamField path="n" type="int">
  Upvalue index (1 is the first upvalue)
</ParamField>

**Returns:** Name of the upvalue, or NULL if index is out of range. Pushes the upvalue's value onto the stack.

**Example:**

```c
// Assuming a function is at index -1
int i = 1;
const char *name;
while ((name = lua_getupvalue(L, -1, i)) != NULL) {
    printf("Upvalue %d: %s\n", i, name);
    lua_pop(L, 1);  // Remove value
    i++;
}
```

### lua_setupvalue

Sets the value of a closure's upvalue.

```c
const char *lua_setupvalue(lua_State *L, int funcindex, int n);
```

<ParamField path="L" type="lua_State*">
  The Lua state
</ParamField>

<ParamField path="funcindex" type="int">
  Stack index of the function
</ParamField>

<ParamField path="n" type="int">
  Upvalue index (1 is the first upvalue)
</ParamField>

**Returns:** Name of the upvalue, or NULL if index is out of range. Pops the new value from the stack.

### lua_upvalueid

Returns a unique identifier for an upvalue.

```c
void *lua_upvalueid(lua_State *L, int funcindex, int n);
```

<ParamField path="L" type="lua_State*">
  The Lua state
</ParamField>

<ParamField path="funcindex" type="int">
  Stack index of the function
</ParamField>

<ParamField path="n" type="int">
  Upvalue index
</ParamField>

**Returns:** Unique identifier (pointer), or NULL if invalid.

### lua_upvaluejoin

Makes the n1-th upvalue of the Lua closure at index funcindex1 refer to the n2-th upvalue of the Lua closure at index funcindex2.

```c
void lua_upvaluejoin(lua_State *L, int funcindex1, int n1, int funcindex2, int n2);
```

<ParamField path="L" type="lua_State*">
  The Lua state
</ParamField>

<ParamField path="funcindex1" type="int">
  Stack index of the first function
</ParamField>

<ParamField path="n1" type="int">
  Upvalue index in first function
</ParamField>

<ParamField path="funcindex2" type="int">
  Stack index of the second function
</ParamField>

<ParamField path="n2" type="int">
  Upvalue index in second function
</ParamField>

## Complete Example: Stack Trace

Here's a complete example of generating a stack trace:

```c
void print_stack_trace(lua_State *L) {
    lua_Debug ar;
    int level = 0;
    
    printf("Stack trace:\n");
    while (lua_getstack(L, level, &ar)) {
        lua_getinfo(L, "nSl", &ar);
        
        printf("  [%d] ", level);
        if (ar.name) {
            printf("%s() ", ar.name);
        } else {
            printf("<unknown> ");
        }
        
        if (ar.currentline > 0) {
            printf("at %s:%d\n", ar.short_src, ar.currentline);
        } else {
            printf("in %s\n", ar.short_src);
        }
        
        level++;
    }
}
```

## Complete Example: Variable Inspector

```c
void inspect_locals(lua_State *L, int level) {
    lua_Debug ar;
    
    if (!lua_getstack(L, level, &ar)) {
        printf("Invalid stack level\n");
        return;
    }
    
    lua_getinfo(L, "nSl", &ar);
    printf("Function: %s\n", ar.name ? ar.name : "<unknown>");
    printf("Source: %s:%d\n", ar.short_src, ar.currentline);
    printf("\nLocal variables:\n");
    
    int i = 1;
    const char *name;
    while ((name = lua_getlocal(L, &ar, i)) != NULL) {
        int type = lua_type(L, -1);
        printf("  [%d] %s = ", i, name);
        
        switch (type) {
            case LUA_TNIL:
                printf("nil");
                break;
            case LUA_TBOOLEAN:
                printf("%s", lua_toboolean(L, -1) ? "true" : "false");
                break;
            case LUA_TNUMBER:
                printf("%f", lua_tonumber(L, -1));
                break;
            case LUA_TSTRING:
                printf("\"%s\"", lua_tostring(L, -1));
                break;
            default:
                printf("<%s>", lua_typename(L, type));
                break;
        }
        printf("\n");
        
        lua_pop(L, 1);
        i++;
    }
}
```
